PENGERTIAN CODING DAN LAINNYA

  1. HTML

    2. HTML adalah bahasa markup standar yang digunakan untuk membuat halaman website dan aplikasi web. Bahasa ini diciptakan oleh Tim Berners-Lee, seorang ahli fisika di lembaga penelitian CERN yang berlokasi di Swiss. HTML digunakan untuk menyusun bagian paragraf, heading, maupun link pada halaman web.

  2. CSS

    3. CSS adalah singkatan dari Cascading Style Sheets. CSS adalah bahasa yang digunakan untuk mengatur tampilan elemen yang tertulis dalam bahasa markup seperti HTML.

    Fungsi dan Manfaat CSS

    CSS berfungsi untuk mengatur berbagai aspek tampilan halaman web, seperti warna, font, layout, dan lainnya. Dengan CSS, Anda dapat mengubah tampilan semua elemen pada halaman web tanpa harus menulis ulang kode HTML. Beberapa manfaat utama CSS adalah:

    Efisiensi: Mengurangi duplikasi kode dan memudahkan pemeliharaan.

    Kustomisasi: Memungkinkan penerapan berbagai macam style pada satu halaman HTML.

    konsistensi>:Memastikan tampilan yang konsisten di seluruh halaman web.

  3. Java script

    4. JavaScript adalah bahasa pemrograman populer yang digunakan untuk membuat website menjadi dinamis, interaktif, dan responsif. Berbeda dengan HTML yang hanya mengatur struktur, JavaScript memungkinkan penambahan logika, animasi, validasi, hingga komunikasi data secara real-time di browser maupun server.
